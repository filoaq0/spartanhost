## What's the Hardware Actually Like?

This isn't the kind of host where "enterprise hardware" is marketing copy. On the VPS side, their premium tier runs AMD Ryzen 9950X (Dallas/Ashburn) and Ryzen 7950X (Seattle) processors with NVMe SSDs in RAID 10 — that's the storage configuration you use when you don't want a single drive failure ruining anyone's day. The budget E5 tier runs dual Intel Xeon E5-2690 chips, which are older but still solid for most workloads.

Network ports on KVM VPS plans are 10Gb/s. You're sharing that bandwidth, so you won't always saturate it, but burst speeds are legitimately fast.
